# Ikea Bilresa Scroll Wheel – Blueprint Home Assistant (FR)

Blueprint pour utiliser la **télécommande Ikea Bilresa Matter avec molette** dans Home Assistant.

Cette télécommande permet de contrôler **3 canaux différents** avec :

- clic
- double clic
- triple clic
- clic long
- maintien
- molette rotative

La molette peut contrôler :

- luminosité
- température de couleur
- teinte de couleur
- volume audio
- mode dynamique
- actions personnalisées

---

# Fonctionnalités

- Support complet de la molette Ikea Bilresa
- Support de **3 canaux indépendants**
- Compatible **lumières et media players**
- Mode **dynamique via input_select**
- Mode **action personnalisée**
- Mode **relaxed** ou **instant**
